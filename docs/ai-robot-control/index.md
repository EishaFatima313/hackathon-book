# AI Robot Control

This section covers how to connect AI agents to ROS 2 robot controllers, enabling intelligent autonomous behaviors.

## Topics Covered

- **AI-Robot Bridge**: Connecting Python AI agents to ROS 2 publishers and subscribers
- **State Management**: Processing sensor data for AI decision-making
- **Command Interpretation**: Converting high-level AI commands to robot movements
- **Reinforcement Learning**: Training AI agents for robot navigation

## Getting Started

Start with our [AI Robot Bridge](/docs/ai-robot-control/ai-robot-bridge) guide to learn the fundamentals of AI-ROS integration.
