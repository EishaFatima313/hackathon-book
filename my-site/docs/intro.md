# Welcome to Robotics Education

This educational platform teaches you about the **Robotic Nervous System (ROS 2)** with a focus on connecting AI agents to humanoid robot control.

## Getting Started

Start with the [ROS 2 Basics](./ros2-basics/ros2-intro.md) chapter to learn fundamental concepts, then progress through the modules to understand how AI agents can control humanoid robots.
